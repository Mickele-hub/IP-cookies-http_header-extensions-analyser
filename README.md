#IP & Cookie Analyzer
Extension de navigateur permettant d'analyser rapidement certaines informations d’un site web :
  -Adresse IP du site
  -Cookies utilisés
  -En-têtes HTTP

Ce projet est conçu comme un petit outil d'analyse web utile pour l'apprentissage de la cybersécurité et de l'inspection des sites web.
Fonctionnalités

Compatible avec :
  -Google Chrome
  -Microsoft Edge
  -Opera

Installation :
  -Télécharger ou cloner le projet
  -Ouvrir le navigateur.
  -Aller dans la page des extensions : chrome://extensions/
  -Activer Mode développeur.
  -Cliquer sur Load unpacked.
  -Sélectionner le dossier du projet.
  -L'extension est maintenant installée.

Utilisation :
  -Ouvrir un site web dans le navigateur.
  -Cliquer sur l’icône de l’extension dans la barre d’outils.
  -L’extension affichera automatiquement :
      .l’adresse IP du site
      .les cookies présents
      .les en-têtes HTTP
